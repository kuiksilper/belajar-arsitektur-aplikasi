# Belajar Arsitektur Aplikasi

Tujuan dokumen ini: mencatat proses belajar coding dan arsitektur aplikasi dari level sederhana sampai kompleks, dengan bahasa yang praktis dan mudah dipahami.

## Cara memakai folder ini

1. Baca `00-roadmap.md` untuk urutan belajar.
2. Setiap selesai belajar satu topik, tulis catatan di `log-belajar.md`.
3. Untuk setiap topik, gunakan format di `template-catatan-belajar.md`.
4. Fokus utama: paham cara aplikasi bekerja, bukan sekadar hafal syntax.

## Prinsip belajar

- Mulai dari aplikasi paling sederhana.
- Pahami alur data: input -> proses -> output -> penyimpanan.
- Gambar arsitektur sebelum coding.
- Setiap konsep harus punya contoh kecil.
- Dokumentasikan dengan bahasa sendiri.
